Man laver en repo på GitHub og derefter tilføjer man sine gruppemedlemmer. For at lave en repo på GitHub, trykker man på "New" og navngiver repoen og trykker create repository. derefter går man ind på sin repo, og tilføjer gruppemedlemmer under "settings" og over under "collaborators" og trykker på "Add people". Så laver man en mappe på sin pc, hvor man kan gemme de filer, man henter fra GitHub og går ind i terminalen og skriver "cd (folder)", så GitHub ved hvilken mappe man henviser til. Så finder man URL inde på GitHub og skriver i terminalen "git clone (URL)", så man ved hvilken fil man henter og arbejder på.

git clone (url) - betyder at når man trykker på "code" inde på github, får man URL, så man kan hente filen lokakt på sin pc.

Efter at man har clonet den ned lokalt på sin pc, skal man ind i terminalen og skrive "git add ." og derefter "commit -m "message" og til sidst "push" så den bliver tilgængelig for de andre du har delt den med.

får at gruppemedlemmerne kan hente den nye fil ned, skal de nu i terminalen skrive "git pull" og de kan så lave ændringer. Når de er færdige med det, skal de igen skrive "git add . " "git commit -m "message" og "git push".
